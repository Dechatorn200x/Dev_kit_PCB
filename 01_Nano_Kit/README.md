# 🛠️ Arduino Nano Integration Kit Board

บอร์ดทดสอบอเนกประสงค์ (Development & Testing Board) สำหรับไมโครคอนโทรลเลอร์ Arduino Nano ที่ออกแบบมาเพื่อรองรับงานระบบควบคุมอุตสาหกรรม, การเชื่อมต่อโครงข่ายเซ็นเซอร์ และการขับโหลดกระแสสูงเบื้องต้น

---

## 📌 แนวคิดสำคัญในการออกแบบ (Design Concepts)

ตัวบอร์ดขับเคลื่อนด้วยสถาปัตยกรรมหลัก **MCU Arduino Nano** โดยรวบรวมบล็อกวงจรที่จำเป็นสำหรับงานอินเตอร์เฟสและควบคุมพื้นฐานไว้อย่างครบถ้วน:

1. **MCU Main Core:** รองรับโมดูลบอร์ดพัฒนา Arduino Nano ติดตั้งบนซ็อกเก็ตเพื่อความสะดวกในการซ่อมบำรุง
2. **Power Supply Stage:** วงจรลดแรงดันไฟเลี้ยงประสิทธิภาพสูงแบบ Buck Converter (LM2596) แปลงไฟเข้ากระแสตรง 12V เป็น 5V สำหรับเลี้ยงระบบหลัก
3. **Isolated Industrial Out (Drive Motor):** มีบล็อกวงจร GPIO OUT ขับโหลดไฟฟ้าแรงดัน 12V (ขับมอเตอร์/รีเลย์) แยกกราวด์ระบบด้วย Optocoupler ป้องกันสัญญาณรบกวนย้อนกลับทำลายชิปหลัก
4. **GPIO IN Stage:** วงจรอินพุตดิจิทัลรับสัญญาณจาก Switch พร้อมชุดตัวต้านทานและตัวเก็บประจุช่วยดักสัญญานป้องกันการเด้งของหน้าสัมผัส (Hardware Debounce)
5. **USB to TTL Interface:** ช่องสัญญาณพอร์ตสื่อสารซีเรียลเพิ่มเติมสำหรับการ Debug ข้อมูลผ่านคอมพิวเตอร์
6. **I2C Protocol Hub:** พอร์ตขยายสัญญาณสำหรับการเชื่อมต่อเซ็นเซอร์ความแม่นยำสูงภายนอก เช่น จอแสดงผล หรือโมดูลวัดค่าต่างๆ

## ⚙️ คุณสมบัติพิเศษด้านการสลับช่องสัญญาณ (Flexible Mode Selection)

เพื่อรองรับการพัฒนาระบบที่มีความยืดหยุ่นสูง บอร์ดนี้จึงได้รับการติดตั้ง **สวิตช์เลือกโหมดทำงาน (Mode Selection Jumper/Switch)** เพื่อจัดการทรัพยากรพอร์ต I2C ดังนี้:

* **I2C to Analog Pins Switching:** สามารถเปิดโหมดสลับสัญญานสายจากบล็อก I2C วิ่งตรงกลับมาเข้าพอร์ตอินพุตอะนาล็อก (Analog Pins) ของ Arduino Nano ได้ทุกช่องสัญญาณ
* **I2C to Digital Pins Switching:** สามารถเปลี่ยนโหมดกำหนดทิศทางสัญญาณ เพื่อวิ่งกลับเข้าใช้งานพอร์ตดิจิทัล (Digital Pins) ของ Arduino Nano ได้ทุกช่องสัญญาณตามความต้องการของซอฟต์แวร์

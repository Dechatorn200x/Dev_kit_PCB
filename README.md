# 🚀 PCB Design Skill Development & Integration Portfolio

คลังเก็บรวบรวมโปรเจกต์การออกแบบแผงวงจรพิมพ์ (PCB Design) 
วัตถุประสงค์หลักเพื่อการศึกษา, พัฒนาทักษะ และทดลองใช้งานจริงในระบบฮาร์ดแวร์ โดยมุ่งเน้นการประยุกต์ใช้งานด้านอิเล็กทรอนิกส์กำลัง (Power Electronics), ระบบควบคุมอุตสาหกรรม (Industrial Automation) และการจัดการสัญญาณสัญญาณรบกวน (Signal Integrity) ผ่านโปรแกรม **KiCad**

---

## 🎯 เป้าหมายการพัฒนาทักษะ (Skill Development Goals)

* **Hardware Topology Learning:** ศึกษาและฝึกฝนการออกแบบวงจรลดแรงดันประสิทธิภาพสูง (Buck Converter), วงจรคัดแยกสัญญาณแบบ Optocoupler และโครงสร้างพอร์ตสื่อสารต่างๆ
* **PCB Layout Best Practices:** พัฒนาทักษะการจัดวางชิ้นส่วนอุปกรณ์ (Component Placement), การเดินลายเส้นทองแดงตามข้อกำหนดขนาดกระแสไฟฟ้า (Trace Width Constraints) และการจัดการระนาบกราวด์ (Ground Plane Area)
* **Design for Manufacturing (DFM):** ฝึกฝนการนำข้อมูลลายวงจรออกในรูปแบบเอกสารการผลิตจริง เช่น ไฟล์ Gerber, Drill files และตาราง BOM (Bill of Materials) เพื่อใช้เตรียมส่งโรงงานผลิต

### 📦 [01] Arduino Nano Integration Kit (`/01_Nano_Kit`)
บอร์ดทดสอบอเนกประสงค์เพื่อฝึกทักษะการวางระบบอินเตอร์เฟสพื้นฐานเข้ากับไมโครคอนโทรลเลอร์ **Arduino Nano**
* **ฟังก์ชันการเรียนรู้:** * ออกแบบระบบ Buck Converter (LM2596) แปลงระดับแรงดัน 12V เป็น 5V 
  * ออกแบบชุดพอร์ตสื่อสาร I2C และวงจร USB to TTL เพิ่มเติมสำหรับงาน Debug
  * ออกแบบบล็อก GPIO Isolated ขับโหลดมอเตอร์กระแสตรง 12V แยกกราวด์ป้องกันสัญญาณรบกวนย้อนกลับด้วย Optocoupler
  * ฝึกทักษะการสลับช่องสัญญาณ (Mode Selection Jumper) ยืดหยุ่นทิศทางสัญญาณพอร์ต I2C วิ่งกลับไปพอร์ตอะนาล็อกและดิจิทัลของบอร์ดควบคุม
